information about characters fro mhe game as well as it's rules. Thie site is hosted on heroku and can be found
at this link: https://overpowerdigitallibrary.herokuapp.com/index.html
The site is built with HTML, Javascript and CSS for styling. This site is hosted on Heroku via PHP.
